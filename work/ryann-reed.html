01 · The work

An in-house video engine for a design-build firm that needed more than highlight reels.

I built a recurring video program for Ryann Reed Design Build — the Pennsylvania residential firm — that treats content like an operations function instead of a one-off project. Each month produces a fresh set of pieces: client testimonials shot on the porch of a just-finished kitchen, team interviews with the project managers and carpenters, before-and-after walk-throughs, and short cuts for the sales team to send during the bid stage.

The point of the program is durability. The same shoot day produces a long-form YouTube cut, a square Instagram set, a vertical TikTok or Reel, and stills the team can drop into proposals. We optimize for the firm's actual sales cycle, which is long and trust-driven, not for chasing views. The result is a library that compounds — every quarter the firm has more proof, more stories, and more ammunition for the next pitch.
