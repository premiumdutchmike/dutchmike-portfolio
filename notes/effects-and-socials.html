B — Portfolio Effects

Hand-picked visual / interaction moves that land on a monochrome editorial layout — Geist type, off-white background, hand-rolled HTML/CSS. No React required for any of these; the JS-flagged ones can run in ~10–40 lines of vanilla.

01

Scroll-pinned diagonal text

A huge editorial word or sentence pinned across the viewport at an angle while the rest of the page scrolls past it. Reads as a typographic anchor between sections.

between-sections difficulty: med uses: position:sticky + rotate(-12deg) 
.pin{position:sticky;top:30vh;transform:rotate(-9deg);font-size:clamp(72px,14vw,220px);letter-spacing:-0.04em}
awwwards SOTD
02

Infinite single-line marquee

One thin row of text or client names scrolling horizontally forever. Used as a page-divider or above the footer — never as the hero.

between-sections difficulty: low pure CSS, no JS 
@keyframes mq{to{transform:translateX(-50%)}} .marquee{display:flex;animation:mq 40s linear infinite}
magicui / marquee
03

Sticky shrinking nav

Top bar starts tall and editorial, compresses to a thin rule + small wordmark after the first 80–120px of scroll. Adds spatial hierarchy without taking real estate.

global / header difficulty: low 1 IntersectionObserver toggles a class 
new IntersectionObserver(([e])=>nav.classList.toggle('shrunk',!e.isIntersecting)).observe(sentinel)
w3 / shrink header
04

Magnetic buttons

Primary CTAs subtly lean toward the cursor inside a generous hover radius. Feels tactile without being cute — keep travel under ~10px.

cta / hero difficulty: med vanilla JS + transform 
btn.onpointermove=e=>{const r=btn.getBoundingClientRect();btn.style.transform=`translate(${(e.clientX-r.left-r.width/2)*0.2}px,${(e.clientY-r.top-r.height/2)*0.2}px)`}
godly.website
05

Noise grain overlay

A fixed-position SVG turbulence filter at ~6% opacity sits over the whole page. Kills the "fresh-from-Figma" digital flatness, gives the off-white print-paper feel.

global difficulty: low SVG feTurbulence, no JS 
<svg style="position:fixed;inset:0;opacity:.06;pointer-events:none"><filter id="n"><feTurbulence baseFrequency=".9"/></filter><rect width="100%" height="100%" filter="url(#n)"/></svg>
godly.website
06

Color-invert on hover

Case-study cards flip from off-white-on-ink to ink-on-off-white when hovered. Strongest version is on the link row itself — no images move, only contrast.

cards / link rows difficulty: low CSS :hover + filter:invert(1) 
.row:hover{background:#111;color:#f6f5f1} .row:hover img{filter:invert(1) grayscale(1)}
awwwards
07

Mask-line text reveal

Headlines reveal by sliding up from inside a clipped box — one line at a time, staggered. Tied to IntersectionObserver so it only fires on first view.

hero / section intros difficulty: med overflow:hidden + translateY 
.line{display:block;overflow:hidden} .line span{display:block;transform:translateY(110%);transition:transform .9s cubic-bezier(.2,.8,.2,1)} .in .line span{transform:none}
framer / mask-text-reveal
08

Blur-in on scroll

Section paragraphs enter slightly out of focus (~8px blur) and resolve as they cross 60% of the viewport. Dreamy without being aggressive.

long-form / case study body difficulty: low CSS filter:blur() + observer 
p{filter:blur(8px);opacity:0;transition:filter .8s,opacity .8s} p.in{filter:none;opacity:1}
reactbits / scroll-reveal
09

Cursor spotlight + invert

A circular mask follows the cursor; inside the circle, the page colors invert. Powerful on a single editorial spread — overused if every section does it.

one hero section difficulty: high mask-image:radial-gradient 
.layer{position:fixed;inset:0;background:#111;color:#f6f5f1;mask-image:radial-gradient(circle 160px at var(--x) var(--y),#000 99%,transparent 100%)}
prismic / css scroll fx
10

Scroll-snap horizontal case study

Inside a single case study, content slides horizontally with scroll-snap-type:x mandatory. Reads like turning pages of a magazine spread.

case study pages difficulty: low pure CSS, no JS 
.deck{display:flex;overflow-x:auto;scroll-snap-type:x mandatory} .deck>*{flex:0 0 100%;scroll-snap-align:start}
awwwards
11

Sticky stacked cards

Each project card pins as it reaches the top, and the next card slides up over it. Great for a tight "selected work" stack of 4–6 items.

work index difficulty: med position:sticky;top:0 per card 
.case{position:sticky;top:0;height:100vh} .case:nth-child(n){--y:calc((var(--i)) * 12px); transform:translateY(var(--y))}
aceternity / sticky reveal
12

Split-letter hover (per-char)

Project titles broken into individual spans; on row hover, characters lift in a tiny rolling wave (10ms-stagger). Subtler than a full marquee, still alive.

work index / links difficulty: med JS splits text, CSS transitions 
title.innerHTML=[...title.textContent].map((c,i)=>`<span style="--i:${i}">${c}</span>`).join('')
30animations
13

Animated underline (variable-width swipe)

Editorial inline links: a 1px rule sweeps in from left, sits for a beat under hover, sweeps out right on leave. Replaces the default browser underline.

body copy / inline links difficulty: low CSS pseudo + transform-origin 
a::after{content:"";display:block;height:1px;background:currentColor;transform:scaleX(0);transform-origin:left;transition:transform .35s} a:hover::after{transform:scaleX(1)}
freefrontend
14

Image-on-link hover (proxy thumb)

Work list is text-only; hovering a row reveals a small project thumbnail that tracks the cursor. Lets the index stay quiet and typographic.

work index difficulty: med fixed-position img follows pointer 
row.onpointermove=e=>{thumb.style.cssText=`left:${e.clientX}px;top:${e.clientY}px;opacity:1`}
godly.website
15

Progress rule (page-scroll indicator)

A 1px black line under the nav fills left-to-right as the page scrolls. Replaces a scrollbar; tells the reader where they are in a long case study.

case study pages difficulty: low CSS animation-timeline:scroll() 
.bar{height:1px;background:#111;transform-origin:left;animation:fill linear;animation-timeline:scroll(root)} @keyframes fill{to{transform:scaleX(1)}}
prismic
16

Image clip-path reveal on scroll

Case-study heroes enter with a thin horizontal band (clip-path inset 49% 0) that opens vertically to full size. Cinematic; uses no opacity fade.

case study hero difficulty: med clip-path:inset() animated 
.hero img{clip-path:inset(48% 0);transition:clip-path 1.1s cubic-bezier(.2,.8,.2,1)} .hero.in img{clip-path:inset(0)}
framer / scroll fx
17

Mix-blend-mode wordmark

The fixed logo uses mix-blend-mode:difference so it flips white on dark sections and black on light ones — no JS, no scroll listeners.

global / header difficulty: low 1-line CSS 
.logo{mix-blend-mode:difference;color:#fff;position:fixed}
godly.website
18

Tabular-numerics ticker

Stats (years experience, projects shipped) animate from 0 → target on first view, using font-variant-numeric:tabular-nums so digits don't dance.

about / stats strip difficulty: low requestAnimationFrame loop 
let n=0;(function tick(){n+=2;el.textContent=Math.min(n,target);if(n<target)requestAnimationFrame(tick)})()
21st.dev
19

Cursor text-swap on link hover

The cursor itself becomes a small word ("view", "open") inside a circle when hovering a project link. Editorial way to remove generic cursor icons.

work index / case links difficulty: med fixed div + pointer:none 
document.onpointermove=e=>cur.style.cssText=`transform:translate(${e.clientX}px,${e.clientY}px)`
awwwards / editorial
20

Section-end caesura (oversized number)

Between sections, a single big tabular numeral ("§ 02") flush-right, all rule and air. No animation — pure typographic punctuation; reads like a print magazine.

between-sections difficulty: low type + spacing, no JS 
.caesura{font-size:clamp(80px,18vw,260px);text-align:right;letter-spacing:-0.04em;line-height:.9;font-variant-numeric:tabular-nums;border-top:1px solid #dcdad2;padding-top:.6em}
